# pspm_sf_scl
## Description
pspm_sf_scl returns the mean skin conductance level for an epoch

## Format
`[sts, scl] = pspm_sf_scl(scr, sr)`

## Arguments
| Variable | Definition |
|:--|:--|
| scr | the input skin conductance response. |
| sr | the input sampling rate. |
| options | the options struct. |

## Outputs
| Variable | Definition |
|:--|:--|
| scl | scl outputs. |
