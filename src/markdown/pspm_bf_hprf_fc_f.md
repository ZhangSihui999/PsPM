# pspm_bf_hprf_fc_f
## Description
pspm_bf_hprf_fc_f is the basis function for heart period response function.

## Format
`[bf p] = pspm_bf_hprf_fc_f(td, soa, p)`

## Arguments
| Variable | Definition |
|:--|:--|
| td | time resolution in second. |
| p | an array with variables (in order) a, b, x0, and A. |
