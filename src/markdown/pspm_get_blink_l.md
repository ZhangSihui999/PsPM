# pspm_get_blink_l
## Description
pspm_get_blink_l is a common function for importing eyelink data (blink_l data)

## Format
`[sts, data]= pspm_get_blink_l(import)`

## Arguments
| Variable | Definition |
|:--|:--|
| import | See following fields. |
| import.data | column vector of waveform data. |
| import.sr | sample rate. |
