# pspm_get_blink_r
## Description
pspm_get_blink_r is a common function for importing eyelink data (blink_r data)

## Format
`[sts, data] = pspm_get_blink_r(import)`

## Arguments
| Variable | Definition |
|:--|:--|
| import | See following fields. |
| import.data | column vector of waveform data. |
| import.sr | sample rate. |
